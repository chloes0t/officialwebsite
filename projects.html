Upcoming Project -- Customer Special Order System

Code snippet from one of the queries:
/* Display full name and phone number of all customers that have ordered products that are arriving on Friday */
/* The query below assumes that the data necessary spans 3 different tables */
SELECT CONCAT(customers.first_name, customers.last_name) AS "to_call", customers.phone_number, orders.product_name
FROM customers
JOIN orders
ON customers.order_id=orders.order_id
JOIN distributors
ON orders.distributor=distributors.distributor_name
WHERE distributors.typical_arrival_date = "Friday";
