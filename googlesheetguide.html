Problem!


The pet store was hosting a charity event and decided to use Google Sheets as a way to record each donation made. Obviously, multiple customers contributed over the course of a day, but how is each individual donation tracked and how can the total be calculated? The long way of doing this is to keep on making new cells or updating the total each and every time a donation is made, which can get messy and cost a lot of time that could be spent doing other things.


Thought process:


I knew how to sum an array of numbers, so I figured I could convert the numerical contents of a single cell into an array, separate them by spaces, and sum them.


I did so by implementing the following formula. (NOTE: This is a demo, not the actual donation sheet. That could not be shown for security reasons.)


=arrayformula(sum(value(split(cell," "))))

Problem!


An event happened that requires us to come up with new prices for products and adding .99 cents to the prices is conventional.


Thought process:


I could use the round function to round the supply price to the nearest tenth and then convert it to a string so I can concatenate it with the appended string ".99".


I did so by implementing the following formula. (NOTE: This is a demo, not the actual sheet. That could not be shown for security reasons.)


=value(concat(rounddown(cell),".99"))

Animal Babysitting Schedule/Cost Calculation


Although we have yet to implement this, I recently made a spreadsheet that stores necessary information about customers and calculates the cost (without tax) of boarding an animal at our store by first calculating the number of nights stayed, and then factoring in additional costs like nail trimmings, playtime sessions, cage setup costs, etc.


NOTE: Names and numbers were made up for this demo for the sake of security.


Formulas used:


=QUERY(BoardingInfo!A$3:G$15,"Select A (cell containing owner's name from info sheet) where C contains '"& [cell containing animal name] &"' ",0)


=datedif(E2,F2,"d")*G2+(H2+I2+J2+K2)
